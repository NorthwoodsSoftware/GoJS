    All built-in arrowhead names for Shape.toArrow and Shape.fromArrow. Use these constants instead of plain strings for autocomplete and type safety.

    Example:

    new go.Shape({ toArrow: go.Arrowheads.Standard })
    // equivalent to:
    new go.Shape({ toArrow: "Standard" })
