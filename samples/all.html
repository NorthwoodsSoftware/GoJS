Extensions

There are three extension directories.

  • extensions/, loadable in a simple <script> tag
  • extensionsTS/, implemented in TypeScript, compiled as a UMD module
  • extensionsJSM/, using go-module.js, compiled as an ES6 module

In all cases you should copy the extension code into your own project and make sure that any require or import is adjusted to fit your environment. Subtle errors can occur if your app loads the GoJS library more than once, especially if they are of different versions.
